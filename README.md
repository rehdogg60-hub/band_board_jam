# band_board_jam

Game Title: Band Board Jam
Brand: Reh Dogg Games Plus
Genre: Casual board game collection (word unscramble, Tic‑Tac‑Toe, memory match)
Theme: Each game is a different live band performance
Players: 1 (solo)

Overview
Band Board Jam is a musical board game collection where three unique bands invite you to play their signature game. Unscramble sacred mantras with Reh Dogg & The Sacred Syllables, battle in rock‑and‑roll Tic‑Tac‑Toe with The Tic‑Tac‑Tonics, and test your memory alongside the Memory Lane Orchestra. Every game feels like a stage show — complete with band banter, custom symbols, and scoring that echoes applause. All under one roof: the legendary Reh Dogg Games Plus label.

The Three Games (Stages)

1. Word Temple – Unscramble peaceful words (SERENITY, RHYTHM, etc.) with hints from the band. Earn “Dogg Points”.
2. Tic Tac Toe – Classic 3‑in‑a‑row, but X is a guitar riff and O is a drum beat.
3. Memory Match – Flip cards to pair eight fictional bands (Reh Dogg, Moon Dogs, Rhythm Saints, etc.).

Why play?

· No timers, no pressure – pure chill brain training.
· Each game has its own visual style and band personality.
· Perfect for music lovers and puzzle fans.
